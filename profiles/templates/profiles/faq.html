Frequently asked questions

{% blockinclude "fragments/faq_card.html" only with question_id="how_to_search" %} {% block question %} How do I search the repository? {% endblock question %} {% block answer %}

Visit our Repository and use the search bar to narrow your search. Keywords should be separated by white spaces and can mix words from different fields (e.g. 'machine learning London' or 'neuron computational modeling'). You can filter results to display only profiles from under-represented countries (list based on the World's Bank list of Low, Lower-Middle and Upper-Middle income economies as of August 2017).

Look at the profile provided by the researcher and Google them for other profiles (e.g. Institution, LinkedIn, ORCID, ResearchGate, GitHub or Twitter). Read the recommendations if any, making sure they are sound (e.g. if Recommender A says 'great talk at OHBM 2017', check the OHBM 2017 program and maybe watch the video if available). Important note: the researchers are solely responsible for their profiles and we cannot guarantee authenticity.

It is a vicious circle: researchers need to be known to get invited as speakers but they need to be invited as a speaker to get known. This cycle can be broken by taking a chance and inviting someone less recognized. Bear in mind that younger scientists also have things to say and they might be more approachable and committed.

{% endblock answer %} {% endblockinclude %} {% blockinclude "fragments/faq_card.html" with question_id="resources" %} {% block question %} What if I don't find a suitable candidate? Are there other resources? {% endblock question %} {% block answer %}

If you cannot identify potential speakers for your event, check out these other great resources:

  • Anne's list, a list of women in systems cognitive neuroscience.
  • Women in Brain Stimulation
  • AcademiaNet, a database of women in all fields of science, nominated by partner agencies.
  • 500 women scientists, a list of women in STEM.
  • BlackInAI, Black researchers in AI, ML and related application areas (health, agriculture, politics, economics, law etc.)
  • LatinXAI, Global Community of LatinX in AI Professionals
  • Queer in AI, an initiative to raise awareness of queer issues in AI/ML
  • BlackInNeuro, an initiative to improve the diversity within neuroscience communities that celebrates and empowers black scholars and professionals in neuroscience-related fields.
  • Queer in Neuro, a network for LGBTQIA+ scientists in Neuro.
{% endblock answer %} {% endblockinclude %} {% blockinclude "fragments/faq_card.html" with question_id="recommend" %} {% block question %} How do I make a recommendation? {% endblock question %} {% block answer %}

Visit our Repository and enter the researcher's name in the search bar or keywords related to your field. Click on 'View Profile' to view the detailed profile of a scientist you would like to recommend. You can then click the “Recommend” button at the top right corner or click the “writing her one” link in the Recommendation section at the bottom of the profile. Anyone can submit a recommendation, disregarding their gender. Recommendations must be positive and use inclusive language. When recommending a researcher, please make sure to include all relevant information, e.g. whether you have seen this person talk at an event before, and where and when that was (as videos of the event might be available). Also include any potential conflicts of interest, disclosing any personal relationship with the scientist (e.g. colleague, close friend, student, etc.). At the moment, it is NOT possible to edit a recommendation, but contact us for related queries.

{% endblock answer %} {% endblockinclude %} {% blockinclude "fragments/faq_card.html" with question_id="create" %} {% block question %} How do I create an account in the repository? {% endblock question %} {% block answer %}

To create a WiNRepo account, click on the button "Sign-up" at the top right of the website. You can choose to create an account by either filling the mandatory fields of Username, Name, Email and Password or through social login via Google, Twitter or GitHub. Once you validate the reCAPTCHA Verification and create your account, an email will be sent to the provided email address in order to activate the account. Please, check your spam box in case you do not see the activation email in your inbox.

{% endblock answer %} {% endblockinclude %} {% blockinclude "fragments/faq_card.html" with question_id="enter" %} {% block question %} How can I add my profile to the repository? {% endblock question %} {% block answer %}

To create a public profile in the repository, you have first to create an account by signing up. For more details on how to create an account, see here. When you activate your account, you will be prompted to confirm whether you identify yourself as a woman and want to create a public profile. You can also choose to continue to your account and edit your public profile later. Make sure that you also agree with our data-storing terms (see here).

{% endblock answer %} {% endblockinclude %} {% blockinclude "fragments/faq_card.html" with question_id="who" %} {% block question %} Who can create an account and profile in the repository? {% endblock question %} {% block answer %}

Anyone can create an account in the repository. Yet, the only feature currently available is the creation of a public profile, if (and only if) you self-identify as a woman active in the research field of neuroscience. More features will be available in the near future through our user accounts and we will be encouraging anyone by then to sign-up. So, stay tuned!

{% endblock answer %} {% endblockinclude %} {% blockinclude "fragments/faq_card.html" with question_id="gdpr" %} {% block question %} What do you do with my information? {% endblock question %} {% block answer %}

We store the data provided (by you) only for display on this website, with the intent to increase the visibility of women in neuroscience. We do not distribute it to third-parties within or outside the EU under any circumstances. You have the right to access your data, as well as for the data to be erased from our database at any time. To do so, simply email us at Women in Neuroscience Repository. To know more about your rights as users of this website, please refer to the European Commission data protection policies.
